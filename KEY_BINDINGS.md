#- Mouse

#- Keyboard
	Up: Moves Map up
	Down: Moves Map down
	Left: Moves Map Left
	Right: Moves Map Right
	
	W: Moves selection(Soldier) Up
	S: Moves selection Down
	A: Moves selection Left
	D: Moves selection Right
	
	R: Reset Zoom
	
	F1: Volume down
	F2: Volume up
	
	F11: Fullscreen on/off
	F12: VSync on/off
	
	M: Next clip
